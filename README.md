# AI Project 4 - Image or Text Recognition (Basic)
Develop a Python-based application capable of recognizing text or objects from images using pre-trained Artificial Intelligence libraries. The system should process image input, extract meaningful information, and display the output accurately.

## Features
- OCR using Tesseract
- Image preprocessing using OpenCV
- Recognition of text from images
- Easy to run in VS Code

## Technologies Used
- Python
- OpenCV
- pytesseract
- NumPy

## Installation
pip install opencv-python pytesseract numpy


## Run
python main.py


## Algorithm
1. Load image
2. Convert to grayscale
3. Apply preprocessing
4. Perform OCR
5. Display output

## Project Outcome
The system extracts text from images and displays the recognized output.
